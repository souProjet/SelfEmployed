
# selfemplyed
### install deno and mysql to get started
##### creates a config.json file in the "class" folder, open it and write this :
```json
{
    "hostname": "<hostname>",
    "username": "<username>",
    "db": "<bdd name>",
    "password": "<password>"
}
```

##### launch the project now with ```deno run -A --unstable --no-check main.ts```
